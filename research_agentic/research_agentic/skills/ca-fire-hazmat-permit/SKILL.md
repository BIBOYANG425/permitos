## Triggers

- Storage, handling, or use of hazardous materials (flammable/combustible liquids, oxidizers, corrosives, toxics, compressed/flammable gases, etc.).
- Aggregate on-site quantity of a hazard class at or above its Fire Code permit amount.
- A project change that introduces or increases hazardous-material inventory.

## Thresholds

| Hazard class (examples) | Expected permit-amount trigger (verify) | Primary source to fetch and quote |
| --- | --- | --- |
| Flammable / combustible liquids | per CFC §105 permit table (class-dependent) | California Fire Code §105 / Chapter 57 |
| Compressed / flammable gases | per CFC §105 permit table | California Fire Code §105 / Chapter 53 |
| Oxidizers / corrosives / toxics | per CFC §105 permit table | California Fire Code §105 / Chapter 50 |

## Exemptions

- Quantities below the Fire Code permit amounts generally do not require the operational permit — confirm the exact class threshold in the fetched permit table.
- This Fire Code permit is distinct from the CUPA HMBP ([[ca-hmbp]]) — a facility can need one, both, or neither; verify each independently.
- Local amendments may lower thresholds or add conditions — the locally adopted code governs.
